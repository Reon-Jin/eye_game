## 🚀 快速启动

```bash
# 1. 创建环境
conda create -n eye_shooter python=3.9 -y
conda activate eye_shooter

# 2. 安装依赖
pip install opencv-python mediapipe numpy

# 3. 进入项目
cd d:\python_object\eye_game

# 4. 测试环境
python test_modules.py

# 5. 启动游戏
python main.py
```

| 按键 | 功能 |
|------|------|
| Space | 开始/暂停/继续 |
| D | 显示/隐藏调试 |
| L | 显示/隐藏特征点 |
| -/= | 调整灵敏度 |
| R | 重置游戏 |
| ESC | 退出 |
